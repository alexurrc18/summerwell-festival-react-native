I built this project as a personal playground to experiment with React Native and Expo. I wasn't familiar with these technologies beforehand, so building this full-stack application from scratch was the perfect hands-on opportunity to explore and learn something new.

## Features
* **Custom Design:** A completely redesigned visual interface tailored to match the specific aesthetic and vibe of the Summer Well festival.
* **Smooth Navigation:** Optimized flow and clear transitions to provide a seamless and natural user experience within the Expo ecosystem.
* **Modern Frontend:** Built efficiently using TypeScript and the Expo tool suite for rapid mobile development and easy testing.
* **Robust Backend:** Powered by a Java Spring Boot REST API to handle data and business logic behind the scenes.
